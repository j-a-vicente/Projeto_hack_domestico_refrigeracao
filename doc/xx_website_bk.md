## Glosário
- **VL** - Valores são os atuais 
- **VC** - cumulativo com base na configuração, valor base os ultimos 30 dias.
- **VCD** - cumulativo DIA contando da 0 hora até 23:59:59
- **SDG** - Sensor do Dreno de Degelo
- **CL** - Cooler 
- **CLHOT** - Cooler ar quente
- **CLCOL** - Cooler ar frio
- **SCDC** - Sensor de Corrente Elétrica DC 
- **SCAC** - Sensor de Corrente Elétrica AC 
- **SCL** - Sensor de Corrente Líquida 
- **MF** - Mosfete 
- **STA** - Sensor de Corrente Ar 
- **STL** - Sensor de Corrente Líquida 
- **SNL** - Sensor de Nível 
- **RL** - Relé 
- **PP** - Pastilha Peltier 
- **DS** - Distribuidor Central 110V 
- **BS** - Base - Régua que recebe energia da DS 
- **DJ** - Disjuntor 
- **NB** - Nobreak 
- **EST** - Estabilizador 
- **MG** - Gerenciador do Rack 
- **BT** - Botão 
- **DT** - Distribuidor - Rede Central de energia do rack 
- **CHAVE** - Chave que controla a fonte de energia que vai alimentar o rack 
- **F** - Fase, lado positivo da corrente elétrica 
- **N** - Neutro, lado negativo da corrente elétrica 
- **B** - Barramento - Tomada de distribuição de energia de cada nível 

## Item do site:

### Itens gerais [Painel Número].
Este item será a media dos sensores 

- VC Consulmo de energia "Watts"
- VC Tempo total do sistema ligado


### KPI.
- VL Temperatura do Hack -10ºc  a 40ºc
- VL Temperatura do liguido de refrigeração  -10ºc  a 100ºc
- VL Nível do reservatório de refrigeração 0 a 100%
- VL Temperatura do refrigerador -10ºc  a 100ºc
- VL Potencia da refrigerador 0 a 100%
- VL Potencia do cooler 0 100%
- VL Potencia de amperagem 0 a 20
- VL Potência máxima Watts 0 a 4400W


### Grafico de Linha evolução VC.

- STA - Sensores de temperatura. 
- SCL - Fluxo do Liguido de refrigeração:
- Consulmo de energia em WATTS.
- SCDC - Sensores de Corrente Elétrica DC
- SCAC - Sensores de Corrente Elétrica AC
- CL - Cooler circulação de ar

### Painel On ou Off
| **MF** | Mosfete |
| **P** | Pastilha Peltier |
| **RL** | Relé |
| **CL** | Cooler |

### Alerta 
- DG - Dreno do Degelo, alerta de entupimento, se acumular água acima do nível de segurança.
- SNL - Sensor de Nível reservatorio do liguido de refrigeração, abaixo do nivel de segurança.
